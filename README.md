# Tool-translate hỗ trợ xử lý file cấu hình JSON (dịch tiếng Trung → tiếng Anh, xuất SoldierCfg ra Excel).

Getting Started
To set up and run this admin dashboard locally, follow these steps:

1. git clone https://github.com/hieu26022002/Tool-translate.git
2. npm install

## 1. Tool dịch JSON (Tiếng Trung → Tiếng Anh)

npx ts-node controller/translate-json.ts

## 2. Tool export SoldierCfg ra Excel (XLSX)

npx ts-node controller/soldier-xlsx.ts
